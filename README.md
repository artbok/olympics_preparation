**Платформа для подготовки к олимпиадам**

Что это такое?

Это приложение, которое помогает школьникам готовиться к олимпиадам интересно. Вместо скучных учебников — интерактивные задачи, соревнования с другими учениками (дуэли) и умная нейросеть, которая может придумывать новые задания.


**Основные возможности:**

Решай: Каталог задач с подсказками.

Соревнуйся: Режим PvP — кто быстрее решит задачу, тот и победил.

Качайся: Система рейтинга (как в шахматах или видеоиграх).

Генерируй: Если задачи закончились, ИИ придумает новые.

Анализируй: Красивые графики твоего прогресса



**Как запустить проект (Docker)**

Мы упаковали весь проект в контейнеры, поэтому для запуска вам не нужно устанавливать Python или Flutter. Достаточно только Docker.

1) Установите и откройте Docker Desktop

2) Скачайте проект и откройте терминал в папке с файлом docker-compose.yml.

3) Запустите проект одной командой:
```
docker compose up -d
```
Эта команда скачает необходимые образы (клиент и сервер) и запустит их в фоновом режиме.

4) Откройте приложение в браузере: Перейдите по адресу: http://localhost:60080
